刑事法律风险防范
万桥拥有一支业务精湛、执业经验丰富的刑事合规法律服务律师团队，万桥刑事合规法律服务团队始终致力于依托专业精湛的执业经验，科学合理地制订防控和管理计划，合理分配企业资源，最终实现对企业刑事法律风险的有效防控和管理。同时也为个人客户提供定制化刑事法律风险服务，让客户增强法律意识和自我监控、自我约束的能力，在刑事风险高发区域构筑警戒线，自觉遵守法律，履行诚信和道德，远离刑事法律风险。
为企业客户提供以下刑事法律风险服务：
1、提供避免贿赂型犯罪（行贿罪，单位行贿罪，单位受贿罪，商业贿赂犯罪、公司企业人员受贿罪等）的法律服务；
2、提供避免财产型犯罪（职务侵占罪，贪污罪，挪用资金罪，挪用公款罪等）的法律服务；
3、提供避免融资类犯罪（如欺诈发行股票、债券罪，违规披露、不披露重要信息罪，非法吸收公众存款罪，集资诈骗罪等等）的法律服务；
4、提供避免财税类犯罪（如偷税罪，抗税罪，逃避追缴欠税、骗取出口退税罪，虚开增值税发票罪等）的法律服务；
5、提供避免生产销售类犯罪（如生产销售伪劣产品罪，生产销售假药罪等）的法律服务；
6、提供避免知识产权类的犯罪（如假冒注册商标罪、假冒专利罪、侵犯著作权罪和侵犯商业秘密罪等）的法律服务；
7、提供避免合同类犯罪（如合同诈骗罪，签订、履行合同失职被骗罪等）的法律服务；
8、提供避免其他类型犯罪（如新领域的网络犯罪、环境保护类犯罪等）的法律服务；
9、协助客户建立完善内部合规制度和风控体系避免以上刑事风险犯罪；
10、协助客户做好刑事风险防控、合规制度构建、法律培训及日常法律咨询、新法快讯等工作；
11、协助应对政府部门调查、突击检查和问询，包括制定预案措施、现场协助和危机管理；
12、针对客户高管人员存在的刑事、合规、反腐败风险，提供紧急法律热线服务，确保客户能在第一时间获得有效法律帮助；
13、根据客户的要求，对客户的交易相对人（供应商），包括相对人本身、股东、实际控制人及高管等开展刑事风控、合规、反腐败调查或审查。
为个人客户提供以下刑事法律风险服务：
1、为客户提供刑事法律风险咨询；
2、协助客户紧急应对相关机构启动的刑事调查，紧急介入并提供应急预案，协助做好舆情监控；
3、就具体刑事案件为客户提供综合刑事辩护服务，进行最优辩护策略决策，协助取得良好的裁判结果；
4、协助客户一并代理刑事附带民事诉讼或关联民事、行政诉讼案件，协助客户进行调查取证及应诉；
5、协助客户与办案机关进行充分沟通，就刑事强制措施、刑事赔偿、刑事和解等事宜进行有效沟通并争取最优结果；
6、协助客户代理刑事与民事存在交叉关系的案件，协助就案件中存在的刑事犯罪线索启动刑事案件立案调查，以刑事案件证据或调查、裁判结果推动民事案件的有利进展。
